## Overview

NexCMS is a custom-built website builder and CMS designed for small businesses and food service operators who need a simple, powerful content management experience without the bloat of traditional platforms.

## Planned Features

- 🏗️ **Visual page builder** — drag-and-drop block-based editor
- 📝 **Content management** — pages, posts, menus, media
- 🔑 **Auth & roles** — owner, editor, viewer permissions
- 🌐 **Custom domains** — one-click publish to custom domain
- 📊 **Analytics integration** — basic traffic and engagement stats

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Vite |
| Backend | Supabase |
| Auth | Supabase Auth |

## Getting Started

```bash
git clone https://github.com/ShadowWalkerNC/NexCMS.git
cd NexCMS
npm install
npm run dev
```
